SmartOCR Pro Portable
=====================

How to run
1. Extract `SmartOCR-Pro-windows-portable.zip` to a writable folder.
2. Keep `SmartOCR Pro.exe`, `README-PORTABLE.txt`, and `resources/ocr` in the same extracted directory.
3. Save your `DeepSeek API Key` in Settings before using AI structuring.
4. Launch `SmartOCR Pro.exe`.

Required files
- `SmartOCR Pro.exe`
- `README-PORTABLE.txt`
- `resources/ocr/runtime/`
- `resources/ocr/models/`

Troubleshooting
- If Windows SmartScreen blocks startup, choose "More info" then "Run anyway".
- If OCR reports missing assets, verify `resources/ocr` was not moved or partially deleted.
- The app depends on Microsoft Edge WebView2. It is preinstalled on Windows 11 and most supported Windows 10 systems.
